# 🌸 Art of Perfume — Quiz de Perfumes

Quiz interactivo que descubre tu perfil olfativo y te recomienda perfumes según tu personalidad.

## 🚀 Cómo publicar en GitHub Pages

1. Crea un repositorio nuevo en [github.com/new](https://github.com/new) (nombre sugerido: `art-of-perfume`)
2. Sube **todo el contenido** de esta carpeta al repositorio
3. Ve a **Settings → Pages → Branch: main → Save**
4. Tu página estará en: `https://TU-USUARIO.github.io/art-of-perfume/`
